## buzz-video-collector セットアップ残作業

### GEMINI_API_KEY の取得と設定
1. https://aistudio.google.com/apikey にアクセス
2. 「Create API key」をクリック
3. 取得したキーを環境変数に設定:
   echo 'export GEMINI_API_KEY="取得したキー"' >> ~/.zshrc
   source ~/.zshrc
4. 動作確認:
   cd /Users/kimuratakuya/line-harness/buzz-video-collector
   python3 -m src --dry-run --no-visual

### cron設定（GEMINI_API_KEY設定後）
buzz-video-collectorのcronを追加:
crontab -e で以下を追加:
0 9 * * * GEMINI_API_KEY="キー" /Users/kimuratakuya/line-harness/buzz-video-collector/run.sh
