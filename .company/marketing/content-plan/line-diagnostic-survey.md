# LINE活用度診断シート

## コンセプト
LINE公式アカウントの活用度を10問で診断。
LINE内で完結し、Lカスタムのサービスデモとしても機能する。

## フロー
リッチメニュー「LINE活用度診断」→ 10問回答 → スコア表示 → レベル別アドバイス → 無料相談CTA

## 診断項目（10問・各1点）

### Q1: あいさつメッセージ
- タイトル: 友だち追加時の「あいさつメッセージ」を設定していますか？
- はい (1点) / いいえ (0点)

### Q2: リッチメニュー
- タイトル: リッチメニュー（画面下のメニュー）を設置していますか？
- はい (1点) / いいえ (0点)

### Q3: 自動応答
- タイトル: キーワードに反応する自動応答メッセージを使っていますか？
- はい (1点) / いいえ (0点)

### Q4: セグメント配信
- タイトル: 友だちの属性や行動に合わせてメッセージを送り分けていますか？
- はい (1点) / いいえ (0点)

### Q5: 配信頻度
- タイトル: 月に2回以上、メッセージ配信をしていますか？
- はい (1点) / いいえ (0点)

### Q6: 登録導線
- タイトル: LINE登録への導線を3箇所以上に設置していますか？（HP、SNS、店舗等）
- はい (1点) / いいえ (0点)

### Q7: 予約・問い合わせ
- タイトル: 予約や問い合わせをLINEで受け付けていますか？
- はい (1点) / いいえ (0点)

### Q8: 登録特典
- タイトル: 友だち追加時にクーポンや特典を用意していますか？
- はい (1点) / いいえ (0点)

### Q9: 効果測定
- タイトル: 配信後の開封率やクリック率を確認していますか？
- はい (1点) / いいえ (0点)

### Q10: ブロック率
- タイトル: ブロック率を定期的にチェックしていますか？
- はい (1点) / いいえ (0点)

## スコア別結果

### 初級（0〜3点）
- レベル: 初級
- タグ: 診断_初級
- メッセージ: もったいない！基本設定だけで反応が変わります
- シナリオ: 基本設定のコツを3日間ステップ配信 → 無料相談CTA

### 中級（4〜6点）
- レベル: 中級
- タグ: 診断_中級
- メッセージ: いい感じ！あと少しの工夫で売上に繋がります
- シナリオ: 活用事例を紹介 → 個別改善提案CTA

### 上級（7〜10点）
- レベル: 上級
- タグ: 診断_上級
- メッセージ: かなり活用できてます！さらに自動化で効率UP
- シナリオ: 自動化・効率化のTips → Lカスタム紹介CTA

## セットアップ手順

### 1. タグ作成
管理画面で以下のタグを作成:
- 診断_初級
- 診断_中級
- 診断_上級
- 診断_完了

### 2. シナリオ作成（3パターン）
各レベルに応じたステップ配信シナリオを作成

### 3. Survey作成（API）
POST /api/surveys で診断を作成し、score_tag_rules を設定:
```json
{
  "name": "LINE活用度診断",
  "onCompleteTagId": "<診断_完了のタグID>",
  "scoreTagRules": "[{\"min\":0,\"max\":3,\"tag_id\":\"<初級タグID>\"},{\"min\":4,\"max\":6,\"tag_id\":\"<中級タグID>\"},{\"min\":7,\"max\":10,\"tag_id\":\"<上級タグID>\"}]"
}
```

### 4. 質問・選択肢登録
各質問をPOST /api/surveys/:id/questions で登録し、
選択肢をPOST /api/surveys/questions/:id/choices で登録（score: 0 or 1）

### 5. リッチメニューに診断ボタン設置
ポストバックアクション: survey:{surveyId} で開始
