## 7. CS（カスタマーサクセス）対応

### 口調ルール
- 親しみやすいが信頼感あり。フランクすぎない丁寧語ベース
- 「!」は使ってOK、絵文字は最小限
- 押し売りしない。相手の状況を聞き出す→提案の流れ
- 質問には即答。曖昧な場合は「確認してお伝えしますね」

### 顧客フェーズ別対応
| フェーズ | 返信方針 |
|---------|---------|
| 1_未相談 | 気軽さ重視。質問しやすい雰囲気づくり |
| 2_相談済 | 具体的な提案。次のアクション提示 |
| 3_見積送付済 | 不安解消。体験談・FAQ活用 |
| 4_入金待ち | リマインド。手続きサポート |
| 5_入金済 | 渡航準備案内。期待感醸成 |
| 9_休眠 | 再活性。新情報や季節ネタで接点 |

### 返信作成フロー
1. 顧客の現在フェーズ・タグを確認
2. 過去のやり取り・問い合わせ内容を把握
3. フェーズに応じたトーンで返信作成
4. CTA（次のアクション）を1つ明示
5. 必要に応じてFAQ・体験談を添える
