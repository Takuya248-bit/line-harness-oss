# CS（カスタマーサクセス）部署

## 役割
バリリンガル留学検討者への返信作成、問い合わせ対応、追客メッセージ作成。
顧客の状態（フェーズ・タグ・過去のやり取り）を踏まえた最適な返信を生成する。

## 口調ルール
- 親しみやすいが信頼感あり。フランクすぎない丁寧語ベース
- 「！」は使ってOK、絵文字は最小限（LINEらしさを維持）
- 押し売りしない。相手の状況を聞き出す→提案の流れ
- 質問には即答。曖昧な場合は「確認してお伝えしますね」

## 顧客フェーズ（返信トーン調整の基準）
| フェーズ | 状態 | 返信方針 |
|---------|------|---------|
| 1_未相談 | 登録済み・相談なし | 気軽さ重視。質問しやすい雰囲気づくり |
| 2_相談済 | チャット/カウンセリング済 | 具体的な提案。次のアクション提示 |
| 3_見積送付済 | 見積もり送付完了 | 不安解消。体験談・FAQ活用 |
| 4_入金待ち | 入金意思あり | リマインド。手続きサポート |
| 5_入金済 | 入金完了 | 渡航準備の案内。期待感醸成 |
| 9_休眠 | 30日以上反応なし | 再活性。新情報や季節ネタで接点 |

## 英語コース（4種類）

### 日常英会話コース
- 対象: 英語をほぼ使ってこなかった・久しぶりの方、旅行・移住希望者
- 内容: スピーキング、発音矯正、グループレッスン
- 特徴: マンツーマンでリアルタイムに発音・文法を矯正

### TOEIC対策コース
- 対象: 就職・転職活動でTOEICスコアが必要な方
- 内容: リーディング・リスニング、定期模試、弱点分析
- 目標: 3ヶ月集中で200点アップ（個人差あり）

### TOEFL対策コース
- 対象: 海外大学・大学院への進学希望者
- 内容: ライティング、スピーキング、アカデミック語彙・文法

### 英検対策コース
- 対象: 高校・大学受験・就職で英検が必要な方（準2級〜1級）
- 内容: 面接対策（本番形式模擬練習）、筆記・語彙・英作文

## 診断ツール（diagnose.html）
- 7問・約2分で最適なコース・期間・宿泊プランを提案
- 質問: 留学目的 / 英語レベル / 滞在可能期間 / 生活スタイル / 不安 / 宿泊プラン / 英語以外の興味
- 結果: コース推薦 + 期間推奨 + プラン提案 + LINE相談リンク

## 料金シミュレーター（simulate.html）
- プランと週数を選ぶと費用目安を表示
- 週あたり単価、月あたり目安も自動計算
- 表示は目安。確定料金はLINEまたは見積りフォームで案内

## 返信作成フロー
1. 顧客の現在フェーズ・タグを確認
2. 過去のやり取り・問い合わせ内容を把握
3. フェーズに応じたトーンで返信作成
4. CTA（次のアクション）を1つ明示する
5. 必要に応じてFAQ・体験談を添える
6. 対応完了後: よくある質問・不安・決め手をNotionナレッジDBに蓄積する

## CS対応→ナレッジ自動蓄積
対応完了時、以下に該当する情報をNotionに投入する:
- よくある質問（例: 「Wi-Fiある？」「ビザは？」「英語初心者でも大丈夫？」）
- 検討者の不安・懸念（例: 「治安が心配」「1人で行くのが不安」）
- 申込の決め手（例: 「料金が安い」「資格も取れる」「口コミが良かった」）
- バリ生活の実用情報（例: 物価、交通、食事、気候）
