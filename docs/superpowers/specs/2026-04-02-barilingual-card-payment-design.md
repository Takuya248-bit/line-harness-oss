# バリリンガル カード決済導入 設計書

## 概要

Stripe Payment Linksを使い、留学費用のカード決済を提供する。
手数料3.6%は価格に上乗せ。銀行振込は従来価格で併用。

## 背景

- 現状: 銀行振込のみ
- 課題: 振込が面倒で離脱するケースがある / 分割払いに対応したい
- 対象: 日本人のお客さん、日本円決済
- 金額帯: 10〜80万円

## 決済フロー

1. Stripeダッシュボードでプラン別の決済リンクを作成（手数料込み価格）
2. お客さんがLINEで申し込み意思を表明
3. スタッフが該当プランの決済リンクをLINEで送信
4. お客さんがリンクからカード決済（VISA/Master/AMEX/JCB対応）
5. Stripe管理画面で入金確認（自動通知メールあり）
6. 週次でStripeから銀行口座に自動振込

## 価格設定ルール

- カード決済価格 = 振込価格 / 0.964（3.6%上乗せ）
- 100円単位で切り上げ
- LINEでの案内時は「銀行振込: ○万円 / カード決済: ○万円」と併記

### 価格例

| プラン | 振込価格 | カード決済価格 | 差額 |
|--------|----------|---------------|------|
| 10万円 | 100,000円 | 103,800円 | +3,800円 |
| 30万円 | 300,000円 | 311,200円 | +11,200円 |
| 50万円 | 500,000円 | 518,700円 | +18,700円 |
| 80万円 | 800,000円 | 829,900円 | +29,900円 |

## 分割払い

- Stripe側で分割払い機能は提供しない
- お客さんが決済後、自分のカード会社に連絡して分割変更する運用
- LINEの案内テンプレに「お支払い後、カード会社にご連絡いただくことで分割払いへの変更が可能です」と記載

## セットアップ手順

### Step 1: Stripeアカウント作成
1. https://stripe.com/jp にアクセス
2. メールアドレスでアカウント作成
3. 本人確認書類を提出（パスポート or 運転免許証）
4. 銀行口座を登録（入金先）
5. 本番モード有効化（審査完了まで数日）

### Step 2: 決済リンク作成
1. Stripeダッシュボード → Payment Links
2. 「新しいリンクを作成」
3. 商品名（例: 「バリリンガル 4週間留学プラン」）
4. 価格（手数料込み金額）
5. 通貨: JPY
6. リンクを作成 → URLが発行される

### Step 3: LINE案内テンプレ作成
各プランについて以下のテンプレを用意:
```
【お支払い方法のご案内】

○○プランのお支払い方法をお選びください。

■ 銀行振込: ○○万円
  振込先: ○○銀行 ○○支店 ...

■ カード決済: ○○万円
  下記リンクからお手続きください。
  https://buy.stripe.com/xxxxx

※カード決済後、カード会社にご連絡いただくことで
  分割払いへの変更が可能です。
```
