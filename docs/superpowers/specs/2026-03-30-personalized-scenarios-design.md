# パーソナライズドシナリオ + ABテスト 設計書

## 概要
診断完了後にカルーセルで「興味のあるプラン」を選ばせ、8パターンのメインシナリオに分岐。
他シナリオはABテスト（CTA最終メッセージのA/B分岐）を適用。

## 全体フロー

```
[友だち追加]
    ↓
[S_01 診断未完了フォロー] ← ABテスト対象（Day5のみ）
    ↓ 診断完了
[診断結果 + プラン選択カルーセル]
    「一番気になるのはどれですか？」
    ↓ 8択から1つ選択
[S_02_X メインシナリオ] ← 8パターンのうち1つ
    ↓ CTA選択 or 完了
[S_03〜S_06, R_01〜R_04] ← S_03/S_04/S_05/R_01はABテスト対象
```

## プラン選択カルーセル（8択）

診断完了時に配信。カルーセル形式で8枚のカードを提示。
各カードのボタンタップで対応タグを付与 → メインシナリオ開始。

| # | カード名 | タグ | 画像テーマ |
|---|---------|------|-----------|
| 1 | 日常英会話 | プラン_日常英会話 | 教室+カフェ |
| 2 | TOEIC・英検・TOEFL対策 | プラン_試験対策 | 勉強+スコアアップ |
| 3 | ワーホリ準備 | プラン_ワーホリ | パスポート+海外生活 |
| 4 | サーフィン×英語 | プラン_サーフィン | サーフィン+ビーチ |
| 5 | ヨガ・ピラティス×英語 | プラン_ヨガ | ヨガ+バリの自然 |
| 6 | 副業スキル×英語 | プラン_副業スキル | PC+カフェワーク |
| 7 | 親子留学 | プラン_親子 | 親子+学校 |
| 8 | まだ決めていない | プラン_未定 | バリ島全景 |

自動応答: 各カードのボタンテキストをキーワードとし、タップ時にタグ付与+シナリオ開始。

## 実装項目一覧

### Phase 1: タグ・カルーセル作成
- プラン選択タグ8個作成
- プラン選択カルーセルテンプレート作成（8枚カード）
- 自動応答8個作成（各プラン名キーワード → タグ付与+シナリオ開始）

### Phase 2: メインシナリオ8本作成
- パターン1-7: 各5通 = 35通のテンプレート新規作成
- パターン8: 既存ABシナリオのA版をコピー（新規作成不要）
- シナリオ8本作成（時刻配信: Day0即時/Day1 06:55/Day2 06:55/Day3 06:55/Day4 20:00）
- 各メッセージにカルーセルCTA（3ボタン: 見積もり/チャット/オンライン相談）

### Phase 3: 診断完了→プラン選択→シナリオ開始の接続
- 診断完了時アクション: プラン選択カルーセルを配信
- 各プランボタンの自動応答: タグ付与+対応シナリオ開始+フェーズ=04+メニュー_02切替

### Phase 4: ABテスト分岐（S_01, S_03-S_05, R_01）
- 既存A用メッセージに3/22_AB_Aタグ絞り込み追加
- B用メッセージを同Day/同時刻で挿入、3/22_AB_Bタグ絞り込み

### Phase 5: 検証
- テスト友だちで8パターン+AB分岐の動作確認
- 配信条件レビュー（タグ条件/タイミング/アクション一致の5項目チェック）
