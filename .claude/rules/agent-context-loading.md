# エージェント作業開始前の必須コンテキスト読み込み

全てのエージェント（メイン・サブ問わず）は、コード変更や意思決定を行う前に以下を必ず読み込むこと。

## 必須読み込みファイル（作業開始前）

1. `.company/secretary/notes/` 内の直近3日分の `*-decisions.md`（意思決定ログ）
2. `.company/secretary/notes/` 内の直近の `*-progress.md`（進捗状況）
3. 作業対象の部署がある場合、その部署の `CLAUDE.md`

## 禁止事項

- decisions.md に記録された決定事項を覆す変更を、ユーザー確認なしに行うこと
- progress.md の「完了済み」に記載された作業を、理由なくやり直すこと
- 過去のログと矛盾する実装を行うこと

## 具体例

- 「Lカスタム」がブランド名と決定済み → 記事やLPで「LINE Harness」を使わない
- ココナラリンク削除が決定済み → CTAにココナラを入れない
- 料金50,000円〜と決定済み → 9,500円の価格表記を使わない
